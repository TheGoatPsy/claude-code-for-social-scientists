# Bundled fonts

This directory ships subset (latin + latin-ext) `woff2` instances of three
variable typefaces, self-hosted so the site makes no third-party font requests.
All three are licensed under the SIL Open Font License, Version 1.1 (see `OFL.txt`).

| Family | Role | Copyright | Source |
|---|---|---|---|
| Newsreader | Display and headings (normal + italic) | Copyright 2019 The Newsreader Project Authors | https://github.com/productiontype/Newsreader |
| Source Sans 3 | Body text | Copyright 2010-2020 Adobe (Reserved Font Name 'Source') | https://github.com/adobe-fonts/source-sans |
| JetBrains Mono | Code and terminal | Copyright 2020 The JetBrains Mono Project Authors | https://github.com/JetBrains/JetBrainsMono |

The `woff2` files were produced from the families' official sources via the
Google Fonts delivery API (latin + latin-ext unicode subsets only) and are
redistributed under the OFL. The full license text is in `OFL.txt`.
